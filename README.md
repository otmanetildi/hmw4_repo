# Homework 4 – Git and GitHub Practice

This repository is part of a Git workflow assignment.
It demonstrates configuring Git, cloning repositories, inspecting Git internals,
using `.gitignore`, and documenting terminal output for grading purposes.

The repository includes a `hwk4` directory containing a detailed log of commands
and results executed throughout the assignment.
